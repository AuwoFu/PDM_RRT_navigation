pcd_preprocess.py
	for create 2D semantic map
	can be run by "python pcd_preprocess.py". Remember to change the location of source file

RRT.py
	Run the RRT algorithm based on given map
	using "python RRT.py -t target -d path_file_location" to run. target means the sellected object name.

navigate.py
	Navigate the agent with given path
	using "python navigate.py -t target" to run with corresponding path file
